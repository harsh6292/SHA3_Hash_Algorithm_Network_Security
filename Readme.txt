Instruction to Run the Project.

1. Make sure Java SDK is installed in the system and the java PATH and CLASSPATH variable is set.
2. Open the command prompt.
3. In the command prompt change the directory to the directory where code is located.
4. Type in the command prompt : "javac SHA3.java" (without double qoutes) to compile the program.
5. Next type in the command prompt : "java SHA3" (without double qoutess) to run the program.
6. The prompt will ask for a string to be entered to calculate its hash.
7. After entering string, the program will display the hash of string in both Binary as well as in Hexadecimal.
